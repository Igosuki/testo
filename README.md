

# BookingNx

This project runs a catering app and api for a restaurant.

### Requirements

- Node 16
- Yarn
- Terraform
- Docker

### Running the dev environment

- ```yarn install```
- ```make infra-dev-up```

#### Navigation
- Mongo admin http://localhost:8081
- React app http://localhost:4200/
- Api http://localhost:3333/

### Testing
```make test```

### Deploying

```make prod-deploy```
